# 杭电OJ算法知识图谱项目总结

## 项目概述

这是一个基于AI的算法题目分析平台，专门用于分析杭电OJ（杭州电子科技大学在线判题系统）的算法题目。该系统能够自动提取题目中的算法标签、数据结构和核心概念，并构建可视化的知识图谱，帮助用户更高效地理解和学习算法知识体系。

## 核心功能

1. **智能题目分析**：利用大语言模型（LLM）自动分析算法题目，提取关键信息
2. **批量分析处理**：支持批量分析题目，自动跳过已分析内容，提高效率
3. **多维度标签体系**：从算法、数据结构和核心概念三个维度对题目进行标注
4. **交互式知识图谱**：可视化展示题目与知识点之间的关联关系
5. **智能题目推荐**：基于知识图谱推荐相关题目，帮助用户系统性学习
6. **多LLM提供商支持**：兼容Ollama（本地）、OpenAI、阿里云通义千问和DeepSeek等多种LLM服务

## 技术架构

### 前端层
- **Streamlit**：构建直观的Web界面，实现交互式操作
- **PyVis**：实现知识图谱的可视化展示
- **HTML/CSS**：定制界面样式，提升用户体验

### 核心层
- **ProblemAnalyzer**：负责调用LLM分析题目内容
- **KnowledgeGraph**：构建和管理知识图谱
- **数据模型**：定义Problem、AnalysisResult和AnalyzedProblem等核心数据结构

### 数据层
- **JSON文件**：存储原始题目数据和分析结果
- **NetworkX**：构建和管理图结构
- **Pandas**：辅助数据处理和统计分析

### 技术栈
- **前端**：Streamlit、PyVis、HTML/CSS
- **后端**：Python、LangChain、NetworkX
- **AI集成**：支持Ollama、OpenAI、阿里云通义千问、DeepSeek
- **数据处理**：Pandas、JSON
- **可视化**：PyVis、NetworkX

## 工作流程

1. **数据准备**：将杭电OJ题目数据整理为JSON格式，包含题目ID、标题、描述、输入输出等信息
2. **模型配置**：选择合适的LLM提供商（如本地Ollama或云端API）
3. **题目分析**：调用LLM分析题目，提取算法标签、数据结构和核心概念
4. **图谱构建**：基于分析结果构建知识图谱，展示题目与知识点的关联
5. **交互应用**：通过Web界面进行题目搜索、标签筛选、知识图谱浏览和相关题目推荐

## 技术亮点

1. **AI驱动的智能分析**：利用大语言模型自动提取题目中的关键算法和数据结构，减少人工标注工作
2. **多模型兼容**：支持多种LLM提供商，既可以使用本地Ollama模型保证隐私，也可以使用云端API获得更好性能
3. **可视化知识图谱**：通过交互式图谱直观展示算法知识体系，帮助用户理解知识点之间的关联
4. **模块化设计**：代码结构清晰，各模块职责明确，便于维护和扩展
5. **用户友好界面**：提供直观的Web界面，支持多种筛选和搜索功能，提升用户体验
6. **高效批量处理**：支持批量分析题目，自动跳过已分析内容，提高处理效率

## 应用价值

1. **算法学习辅助**：帮助学生系统学习算法知识，理解知识点之间的关联
2. **题目分类管理**：为算法题目建立结构化的标签体系，便于检索和管理
3. **学习路径推荐**：基于知识图谱推荐相关题目，构建个性化学习路径
4. **教学资源整合**：为教师提供算法教学的辅助工具，优化教学资源配置
5. **算法竞赛备战**：为参加算法竞赛的学生提供题目分析和推荐功能，提高备战效率

## 项目创新点

1. **AI与算法教育的结合**：利用大语言模型自动分析算法题目，为算法教育提供智能辅助工具
2. **知识图谱的构建与应用**：将算法题目与知识点关联起来，形成可视化的知识网络
3. **多LLM提供商的集成**：兼容多种LLM服务，提供灵活的选择空间
4. **交互式学习体验**：通过Web界面提供直观的交互方式，提升学习体验
5. **模块化与可扩展性**：代码结构模块化，便于功能扩展和技术升级

该项目展示了如何将人工智能技术应用于算法教育领域，为算法学习和教学提供了新的工具和方法。通过智能分析和可视化展示，帮助用户更高效地理解和掌握算法知识体系。
