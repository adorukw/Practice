🚀 C语言HTTP服务器

这是一个完全用C语言编写的HTTP服务器，仅使用标准库和系统调用！

⚡️ 高性能

直接使用系统调用，无额外开销，极致性能。

🔧 原生实现

仅使用C标准库和POSIX API，无需第三方库。

📁 文件服务

支持静态文件服务和自动目录列表。

🔄 多线程

每个连接独立线程，支持高并发。

🛠️ 服务器信息

  • 端口：8080
  • 协议：HTTP/1.1
  • 线程模型：每连接一线程
  • 支持文件类型：HTML, CSS, JS, 图片等

🔍 功能演示

🏠 首页 📁 静态文件夹 ❌ 404页面 🌐 外部链接

尝试在static文件夹中创建文件，服务器会自动显示。

💻 技术栈

这个服务器使用了以下技术：

  • socket() - 创建网络套接字
  • bind() - 绑定端口
  • listen() - 监听连接
  • accept() - 接受连接
  • pthread - 多线程处理
  • open()/read() - 文件操作
