# 口袋妖怪风格个人博客需求文档

## 1. 项目概述

### 1.1 项目背景

创建一个采用GBA时期《口袋妖怪 火红/绿宝石》视觉风格的个人博客网站，将博客内容以游戏化、像素艺术化的方式呈现。网站的核心目标是打造一个兼具强烈怀旧感、游戏沉浸感的个人内容发布与分享平台，让技术分享、生活记录、个人思考等博客内容通过口袋妖怪式的界面和交互方式进行展示。

### 1.2 项目目标

- **短期目标**：建立基础博客框架，支持文章发布、分类展示、评论功能
- **中期目标**：实现完整的博客管理功能，包括标签系统、文章搜索、阅读统计
- **长期目标**：加入社交互动元素和个性化功能，如用户订阅、文章收藏、主题切换

### 1.3 核心功能

- **文章内容管理**：支持Markdown格式文章发布与编辑
- **分类与标签系统**：多级分类和标签管理
- **响应式设计**：桌面/移动端适配
- **文章详情页**：图文混排展示，支持代码高亮
- **阅读状态标记**：文章阅读进度追踪
- **文章归档**：按时间、分类、标签归档浏览
- **搜索功能**：全文检索和筛选搜索

### 1.4 目标用户

- **主要用户**：博客作者本人，用于个人内容创作与管理
- **次要用户**：对技术、生活、游戏文化感兴趣的读者
- **使用场景**：个人知识分享、经验记录、项目展示、日常随笔发布

## 3. 技术实现需求

### 3.1 技术栈

```
前端：
- 框架：Vue 3 + Composition API
- 构建工具：Vite
- 样式方案：Tailwind CSS + 自定义像素样式组件
- UI组件库：自定义像素风格Vue组件
- 状态管理：Pinia
- 路由：Vue Router
- Markdown渲染：marked + highlight.js
- 图标库：自定义像素图标或Lucide图标像素化

后端：
全栈应用
- 后端框架：Express.js或NestJS
- 数据库：PostgreSQL + Prisma ORM
- 部署：个人虚拟机 + Docker
- 内容管理：自行开发管理后台
```

## 7. 设计规范细节

### 7.1 组件设计规范

1. **按钮组件**
   - 主要按钮：红色背景，白色文字，深色边框
   - 次要按钮：蓝色背景，白色文字
   - 文字按钮：黄色文字，下划线效果
   - 悬停效果：按钮上浮1-2像素，阴影加深
   - 点击效果：按钮下沉，阴影减少

2. **卡片组件**
   - 边框：4px深色边框
   - 背景：白色或浅色背景
   - 阴影：右下角阴影，模拟凸起效果
   - 悬停：轻微上浮，边框颜色加深

3. **对话框/弹窗**
   - 边框：双层边框，内浅外深
   - 背景：半透明白色叠加网格背景
   - 关闭按钮：右上角"×"像素图标
   - 动画：从中心放大出现

4. **表单元素**
   - 输入框：凹陷效果，获得焦点时边框高亮
   - 复选框/单选框：自定义像素风格
   - 下拉菜单：游戏内菜单样式
   - 标签：不同颜色表示不同分类

### 7.2 动画与过渡

1. **页面过渡**
   - 页面切换：淡入淡出 + 像素化过渡
   - 加载动画：精灵球旋转
   - 滚动效果：视差滚动，不同层级不同速度

2. **微交互**
   - 链接悬停：下划线展开，颜色变化
   - 按钮点击：下沉效果 + 点击音效（可选）
   - 表单焦点：边框发光，像素闪烁
   - 消息提示：从底部弹出，游戏内提示框样式

## 8. 内容管理与发布流程

### 8.1 本地开发工作流
```
1. 创建新文章
   $ npm run new "文章标题"

2. 本地编写
   - 在content/posts/下编辑Markdown文件
   - 使用本地预览服务器实时查看
   - 添加图片到public/images/

3. 构建测试
   $ npm run build
   $ npm run preview

4. 部署发布
   - 提交到Git仓库
   - 自动触发CI/CD部署
```

### 8.2 基于Git的内容管理
